An API to connect BuddyPress to Foursquare.

== Description ==

This plugin is a library to connect your BuddyPress powered website to the XML or JSON Foursquare API.

== Installation ==

1. Upload `bp-foursquare-api` folder to `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Create a UI for your website
1. Smile at your achievement
